# Suffix

My personal website at https://www.suffix.be.

## Development

### Nix

Make sure to have nix with flakes enabled.

1. Clone the repository `git clone git@github.com:cimm/suffix.git`
2. In the directory `cd suffix` run `nix develop`
3. Start a Jekyll sever in the new shell `jekyll serve`
4. Commit changes `git commit` and deploy to production `rake deploy`

### Ski Maps

The /ski page displays map thumbnails for each ski resort. The images are generated by the `script:locationthumb` QGIS script. To enable this, the script must first be loaded into QGIS: open the “Processing Toolbox” in QGIS, right-click “Scripts”, select “Create new script”, and paste the code. Once installed, the script can be executed manually within QGIS or via the command line using `qgis_process`. For the Jekyll site, the SkiMapTag automatically handles the `qgis_process` execution, passing the specific coordinates and output path for each resort.

To run manually: `qgis_process run script:locationthumb --CENTER_LAT=47.40 --CENTER_LNG=13.39`
